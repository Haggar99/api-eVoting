# api-eVoting


1. `installing nodejs`
Install [NodeJs](https://nodejs.org/en/download/) version > 10.

2.`clone source file from git `
```bash
git clone https://github.com/Haggar99/api-eVoting.git
```
3.
```bash
cd api-eVoting
```

4. `install all packages`
```bash
npm install
```

6. `start server`
```bash
nodemon
```
